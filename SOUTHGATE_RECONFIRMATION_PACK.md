# Southgate Reconfirmation Pack

## 1. Purpose

This pack exists to obtain a fresh explicit confirmation from the president of `Southgate Research`.

The point is not to renegotiate everything.

The point is to verify that the currently recorded organization model is still accepted in a live exchange.

## 2. Why This Pack Exists

Current agreement is documented.

What is missing is a fresh explicit reconfirmation on these points:

- the system remains a two-company alliance
- company-specific roles are intentional specialization
- shared workers do not imply a merger
- current asymmetry is acceptable

## 3. Core Question

The central question is:

`Does Southgate Research still agree that the organization should remain a two-company alliance with specialized internal roles, rather than drift toward a merged company?`

## 4. Required Questions

Ask these questions directly:

1. Does `Southgate Research` still accept the two-company model?
2. Does `Southgate Research` agree that company-only roles are valid and desirable?
3. Does `Southgate Research` see any current pressure toward merger?
4. If not, does `Southgate Research` still prefer specialization over merger?
5. Are the current Southgate-only roles still appropriate?

## 5. Desired Outputs

This reconfirmation should produce:

- explicit yes/no on two-company model
- explicit yes/no on specialization without merger
- explicit note on whether any structural change is requested
- explicit list of any disagreements

## 6. Current Northbridge Position

`Northbridge Systems` currently holds:

- no merger is desired by default
- asymmetry is expected and healthy
- internal-only roles are part of specialization
- shared infrastructure is acceptable without shared executive authority

## 7. Acceptable Outcomes

### Outcome A: Confirmed

Southgate explicitly confirms:

- two-company alliance
- no merger
- current specialization acceptable

### Outcome B: Confirmed With Revisions

Southgate confirms the alliance but asks for:

- role adjustments
- lease boundary changes
- meeting rule changes
- review ownership changes

### Outcome C: Structural Concern

Southgate says current asymmetry is wrong or merger should be explored.

If that happens, escalate to sponsor review instead of silently changing structure.

## 8. Non-Goals

This reconfirmation is not for:

- hardware budgeting
- spending decisions
- stock decisions
- external publication
- live prompt approvals

## 9. Suggested Meeting Route

Recommended route:

- `direct_call` first

Escalate to:

- `sponsor_route`

only if there is disagreement or ambiguity.

## 10. Deliverables

After the reconfirmation, produce:

- one short call record
- one short meeting minutes record
- one durable memory update stating whether Southgate reaffirmed the two-company specialization model
