## 6. Typical Selection Guide

Use:

- summary
  - when the sponsor wants the shortest clean offer description
- profile
  - when a profile page or marketplace bio is needed
- post variants
  - when one short social/public post is needed
- service page
  - when a full page block or landing-page section is needed

## 7. Publication Sequence

Recommended order:

1. choose the exact asset
2. confirm the matching approval sheet
3. run the public-offer checklist
4. confirm the exact final version with the sponsor
5. only then publish
