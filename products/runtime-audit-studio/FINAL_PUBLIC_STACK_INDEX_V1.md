# Runtime Audit Studio Final Public Stack Index V1

## 1. Purpose

This document is the index for the current public-facing asset stack of `Runtime Audit Studio`.

It exists so the sponsor can see, in one place:

- which asset to use
- what each asset is for
- what approval sheet applies
- what is still gated

## 2. Master Rule

No asset in this stack is auto-approved for public release.

Use this index to find the right draft and the right gate before publishing anything.

## 5. Typical Selection Guide

Use:

- summary
  - when the shortest clean offer description is needed
- profile
  - when a profile page or marketplace bio is needed
- outreach
  - when one short outbound message or proposal opener is needed

## 6. Publication Sequence

Recommended order:

1. choose the exact asset
2. confirm the matching approval sheet or gate
3. keep the exact final use sponsor-gated
4. only then publish or send
