## What Is Commercially Ready

Commercially ready for sponsor-controlled use:

- one bounded first offer
- intake and quote structure
- a sample quote
- a short public summary
- a short public profile
- marketplace-oriented listing material
- platform-specific variants for Upwork, Contra, LinkedIn, and direct outreach

## What Is Not Yet Auto-Approved

Still gated:

- actual marketplace posting
- actual profile publication
- exact platform selection
- any edit that widens the current scope
- any wording that implies implementation or production certification

## Current Safe Edit Rule

Allowed without rebuilding the whole packet:

- shortening
- trimming for length limits
- keyword adjustments
- headline alternatives
- CTA tightening

Not allowed without fresh sponsor review:

- stronger reliability claims
- autonomy claims beyond current evidence
- hidden implementation language
- broader scope than the current audit offer

## First Commercial Use Recommendation

Recommended first path:

1. keep `Snapshot Runtime Audit` as the default first package
2. use `PUBLIC_SUMMARY_V1.md` as the shortest external explanation
3. use `PROFILE_DRAFT_V1.md` or `PLATFORM_LISTING_VARIANTS_V1.md` when a marketplace/profile surface is required
4. keep `OUTREACH_COPY_V1.md` as support, not the primary public anchor

## Post-Launch Roadmap

After the first bounded commercial loop, Northbridge should not jump straight into implementation.

Recommended progression:

1. repeat `Runtime Audit Studio` enough times to prove the offer shape
2. incubate `Worker Patch Ops Studio` as the next evidence-first product
3. hold `Technical Brief Studio` behind explicit quality gates
4. defer automation implementation to a later sponsor-approved pilot

Roadmap reference:

- `products/runtime-audit-studio/NEXT_PRODUCT_ROADMAP_V1.md`

## Cross-Sell Boundaries

Safe cross-sell from an audit:

- follow-up audit
- bounded worker evaluation
- approval-gated patch proposal

Not safe to imply from an audit sale:

- implementation included
- runtime guarantee
- autonomous remediation
- production ownership transfer

## De-Risking Milestones Before Expanding Scope

- 3 paid external audit deliveries
- explicit client acceptance of bounded scope
- no fresh `ready_for_external_send: no` failures on the next promoted product
- guarded runtime path stays the default
- evidence reproducibility remains part of every delivery

## Current Honest Sales Position

Northbridge is strongest when it sells:

- evidence-first runtime review
- bounded operational diagnosis
- honest next-fix prioritization

Northbridge is not yet strongest when it sells:

- open-ended implementation
- live intervention
- sweeping autonomy claims
- polished freeform AI consulting language without evidence

## Recommended Sponsor Decision

Recommended decision:

- approve this packet as the current commercial baseline
- keep exact external publication sponsor-gated
- use this packet as the source of truth when choosing what to publish or sell first
- use the roadmap only to sequence later product expansion, not to widen the first offer prematurely
